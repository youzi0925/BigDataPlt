沙特成为第一个向机器人授予公民资格的国家


来源：新浪科技 发布于2017/10/31

据外媒报道，未来展现技术的进步， 沙特阿拉伯成为了世界上第一个授予机器人公民身份的国家。这个称为“索菲亚”（Sophia） 的表情机器人由香港Hanson Robotics公司研发，此前已经有过几次公开露面。 索菲亚近日出现在沙特阿拉伯的未来投资计划会议上，并接受了CNBC Andrew Ross Sorkin的采访。

此事件随后在中外大量社交平台上引起轩然大波，并在网络上给用户带来机器人将取代人类的反AI恐慌。
